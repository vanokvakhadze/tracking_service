# დაიწყე

ეს guide-ი 5 წუთში გადავლის ყველაფერი რაც გჭირდება პირველი თანამშრომლის ცვლის ავტომატური აღრიცხვისთვის.

## 1. შექმენი ანგარიში

გადადი [trackpro.ge](/) → **დაიწყე უფასოდ** → შეიყვანე ელფოსტა + პაროლი.

**კომპანიის ინფო:**

- **სახელი** — როგორ ჩანდება შენი workspace-ი (მაგ. "გლოვო ლოგისტიქა")
- **Subdomain** — `[სახელი].trackpro.ge` (auto-suggested, შემდეგ შეუცვლელი)
- **დროის ზონა** — დანაგულისხმევი `Asia/Tbilisi`

14 დღიანი უფასო ცდა იწყება ავტომატურად. ბარათი არ მოითხოვება.

## 2. დაამატე პირველი ლოკაცია

Admin-ის dashboard-ში → **ლოკაცია** → **+ ახალი**.

ორი მეთოდი:

- **რუკაზე** — გადათრევით pin-ი
- **მისამართით** — `ვაჟა-ფშაველას 76` ჩაწერე → Mapbox-ი იპოვის

**მნიშვნელოვანი ფილდები:**

- **სახელი** — როგორ ვუყურებთ მენეჯერი (მაგ. "ცენტრალური ოფისი")
- **კატეგორია** — Office / Client Site / Warehouse / Checkpoint / Other
- **Trigger რადიუსი** — 50–1500 მ. აქ შემოსვლა → SHIFT_START
- **Boundary რადიუსი** — 100–5000 მ. გასვლა → alert ადმინს

📖 დეტალები: [ლოკაცია + გეოფენსი](/docs/locations).

## 3. დაამატე გუნდი

**გუნდი** ნავიგაცია → **+ თანამშრომელი** → შეიყვანე ელფოსტა + role.

- **employee** — ნახულობს მხოლოდ თავის ცვლას
- **tenant_admin** — ნახულობს მთელ კომპანიას, ქმნის ლოკაციებს
- **super_admin** — მხოლოდ Sazeo platform-ისთვის

Invite link-ი იგზავნება ელფოსტით. თანამშრომელი დაარეგისტრირდება + ჩამოტვირთავს მობილური აპლიკაციას.

## 4. მობილური აპლიკაცია

თანამშრომელი:

1. ჩამოტვირთავს **TrackPro** App Store ან Google Play-დან
2. Login → იღებს permission requests:
    - **Location: Always** ← მნიშვნელოვანი! Background tracking-ისთვის
    - **Notifications** — ცვლის push-ისთვის
    - **Camera** — provisional location-ის ფოტოებისთვის
3. **მთავარი** screen → "ცვლა მზადაა — გადადი სამუშაოზე"

📖 დეტალები: [მობილური აპლიკაცია](/docs/mobile-app).

## 5. ცვლა იწერება ავტომატურად

თანამშრომელი მიდის ლოკაციაზე → Trigger zone-ში გადააბიჯებს → **30 წამიანი hysteresis** → push: "ცვლა დაიწყო".

გადახვალს → 60 წამიანი hysteresis → push: "ცვლა დასრულდა".

შენ admin dashboard-ში ცოცხლად ნახულობ:

- რომელი თანამშრომელი არის რომელ ლოკაციაზე
- რამდენი დრო შრომის
- ცოცხალი GPS pin რუკაზე
